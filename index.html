Strategic AI Integration Assessment

The Ends · Ways · Means strategy pipeline

A small system that turns one business's intake into a deck-ready analysis, then renders it into an 18-slide AI integration strategy. Objectives first; AI named only where it moves a measurable metric.

ENDS — what to achieve WAYS — how to achieve it MEANS — resources, incl. AI
The pipeline

From a business to a deck, in five moves

Each stage has one job. The analysis stays pure content; the generator owns the visual system.

01 · INPUTIntake
The business: overview, ranked objectives, department-by-department current state.
02 · ENGINEThe model
Prompt + scoring rubrics apply the framework and enforce the guardrails.
03 · DATAanalysis.json
The analysis as structured data, conforming to schema.json.
04 · RENDERindex.js
Data-driven builder — palette, layouts and color logic for all 18 slides.
05 · OUTPUTThe deck
An 18-slide .pptx strategy, ready to present or refine.

Run it

Three things to build a deck

You need index.js, an analysis.json, and the Node dependencies. Then one command.

# 1 · install dependencies (once)
npm i pptxgenjs react-icons react react-dom sharp

# 2 · produce analysis.json — via the prompt in the model doc, or by hand against schema.json

# 3 · render the deck
node index.js analysis.northwind.json "AI Strategy - Client.pptx"

# for a real client, set  meta.is_demo: false  to drop the demo labels
How the model thinks

Seven rules the analysis obeys

  1. 1Objectives first, AI last. Define the Ends, fix the Way, then name AI as a Means.
  2. 2Every recommendation traces to a ranked objective. No orphan tooling.
  3. 3AI only where it moves a metric. Each opportunity carries a measurable benefit.
  4. 4Augmentation over replacement. Humans keep judgment, relationships, exceptions.
  5. 5Incremental and milestone-based. Quick win → medium-term → long-term, each with a gate.
  6. 6Honest about readiness and risk. Rate low where it's low; name the gating constraint.
  7. 7Consulting-grade. Specific, quantified, plain-spoken — written for this business.
